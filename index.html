smart팀 졸업작품 소개

작품명 : smart shop

개발환경 : html / css / javascript / adobe XD

작품 소개 : 노트북을 주로 판매하는 인터넷 쇼핑몰 웹사이트입니다.

작품의 특징 : 노트북을 주로 판매하는 인터넷 쇼핑몰 브랜드를 만들어서 인터넷 쇼핑몰의 필수적인 기능과 노트북 쇼핑몰에서 필요한 기능을 둘 다 구현했습니다.

작품 기획 의도 : 인터넷 쇼핑몰을 이용하는 고객의 편의를 최대한 충족시킬 수 있도록 고객이 인터넷 쇼핑몰에서 원하는 페이지와 기능을 중심으로 해서 만들었습니다.

작품 테마 색 : 보라색, 연보라색

주요 페이지

(1) 메인 페이지 : 이벤트 슬라이드 배너와 브랜드별 주요 상품 홍보관을 배치하여 고객이 웹사이트에 접속하자마자 쇼핑몰의 이슈를 접하실 수 있도록 만들었습니다.

(2) 제품 페이지 : 제품을 비즈니스, 크리에이터, 게이머 분류로 분류하여 고객님께서 원하는 용도에 맞는 스펙을 가진 노트북을 쉽게 찾으실 수 있도록 만들었습니다.

(3) 다운로드 : 노트북에 필요한 소프트웨어를 다운로드받을 수 있는 페이지입니다.

(4) 고객지원 : 온라인 상담, AS 문의, 자주 묻는 질문 페이지를 배치한 고객의 질문에 답변하는 것이 주 업무인 곳입니다.

(5) 오프라인 스토어 : 오프라인 스토어에 대한 문의와 오프라인 매장 상담 예약을 할 수 있는 오프라인 스토어 관련 페이지입니다.

(6) 마이페이지 : 장바구니, 주문내역 확인 같은 기본적인 기능이 위치한 곳입니다.
